Projects

Warbler

Warbler is a serverside Twitter clone built with Flask, WTForms, PostgreSQL, SQLAlchemy, Jinja templating, jQuery, and Axios. I implemented bug fixes to the user profile, logout, and homepage routes, and added features so a user can like a Warble(tweet). A user of Warbler can: write posts, like other's posts, edit their profile, follow other user's, and delete their profile. I was able to implement well tested, high quality code to a larger, existing codebase.

Jobly

Jobly is a full stack app Indeed clone! It's built using Node/Express with PostgreSQL on the backend and React on the frontend. Users can register/login with their account credentials. Logged in users can browse and filter companies and view jobs to apply for. I used test driven development (TDD) on the backend and designed and implemented the component heirarchy for the frontend.
